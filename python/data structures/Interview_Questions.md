# Data Structures - Interview Questions

1. **Does list maintain insertion order?**
   *Answer*: Yes, lists in Python are ordered collections and they maintain insertion order.

2. **Can we access list items with negative index?**
   *Answer*: Yes, Python supports negative indexing. For example, `-1` refers to the last element, `-2` refers to the second to last, and so on.

3. **Which function empties the list?**
   *Answer*: The `clear()` method removes all items from a list, emptying it.

4. **Which class represents the group of characters?**
   *Answer*: The `str` (string) class represents a sequence or group of characters in Python.

5. **What is slicing?**
   *Answer*: Slicing is a technique used to extract a specific portion or sub-sequence from a sequence type (like strings, lists, or tuples). It uses the syntax `sequence[start:stop:step]`.

6. **Which function toggles the cases?**
   *Answer*: The `swapcase()` method for strings toggles the cases, converting uppercase characters to lowercase and vice versa.

7. **Does set take duplicate elements?**
   *Answer*: No, sets do not allow duplicate elements. They automatically remove any duplicates and only store unique values.
