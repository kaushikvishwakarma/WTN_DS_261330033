# Functions, Modules and Packages - Interview Questions

1. **List few built in functions.**
   *Answer*: `print()`, `len()`, `type()`, `sum()`, `max()`, `min()`, `abs()`, `range()`.

2. **Which keyword marks the start of the function?**
   *Answer*: The `def` keyword marks the start of a function definition in Python.

3. **Which keyword ends the execution of a function?**
   *Answer*: The `return` keyword ends the execution of a function and optionally passes back a value to the caller.

4. **Which allows you to logically organize your python files?**
   *Answer*: **Modules** (single `.py` files) and **Packages** (directories containing multiple modules and an `__init__.py` file) allow you to logically organize Python code.

5. **How to import a module with alias name?**
   *Answer*: You can use the `as` keyword. For example: `import pandas as pd`.

6. **Which command lists the built in modules?**
   *Answer*: You can use `help('modules')` in the interactive shell or import `sys` and check `sys.builtin_module_names`.

7. **Can we import multiple modules with single import statement?**
   *Answer*: Yes, you can import multiple modules by separating them with a comma. For example: `import os, sys, math`.
