# Python Fundamentals - Interview Questions

1. **List out few scripting languages.**
   *Answer*: Python, JavaScript, Ruby, Perl, Bash.

2. **What is the another name for Python 3.0?**
   *Answer*: Python 3000 or Py3k.

3. **Which command enables the interactive terminal mode?**
   *Answer*: Running `python` or `python3` in the command line/terminal without any script name.

4. **List out few mutable data types.**
   *Answer*: Lists, Dictionaries, and Sets.

5. **Which function returns the location of an object?**
   *Answer*: The `id()` function returns the memory address (identity) of an object.

6. **Which operator does integer division?**
   *Answer*: The floor division operator `//`.

7. **What is type casting?**
   *Answer*: Type casting is the process of converting a variable from one data type to another. For example, converting a float to an integer using `int()`.

8. **How to do multi line comment?**
   *Answer*: By using triple quotes (`'''` or `"""`) at the beginning and end of the block of text, or by prefixing each line with a `#` symbol.

9. **Which built in function returns a sequence of integers?**
   *Answer*: The `range()` function.

10. **Can we use else with for?**
    *Answer*: Yes, in Python we can use an `else` block with a `for` loop. The `else` block executes when the loop terminates normally (meaning it wasn't interrupted by a `break` statement).

11. **Can we write statements after break keyword?**
    *Answer*: Any statements written immediately after a `break` keyword within the same block will not be executed because `break` immediately terminates the loop execution.

12. **Which statement does nothing?**
    *Answer*: The `pass` statement.
