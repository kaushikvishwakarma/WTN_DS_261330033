# Matplotlib and Seaborn - Interview Questions

1. **What are the data visualization techniques used for categorical data?**
   *Answer*: Common data visualization techniques for categorical data include:
   - **Bar Charts**: To compare quantities corresponding to different groups.
   - **Pie Charts**: To show the proportion of categories within a whole.
   - **Count Plots**: A specific type of bar plot that shows the counts of observations in each categorical bin.
   - **Point Plots**: To show point estimates and confidence intervals for categorical groups.

2. **What are the data visualization techniques used for numerical data?**
   *Answer*: Common data visualization techniques for numerical data include:
   - **Histograms**: To visualize the distribution of a continuous variable by grouping it into bins.
   - **Box Plots (Whisker plots)**: To display the distribution of data based on a five-number summary and identify outliers.
   - **Scatter Plots**: To show the relationship and correlation between two continuous variables.
   - **Line Charts**: To display trends over time (time series data).
   - **Density Plots (KDE)**: A smooth, continuous version of a histogram to estimate the probability density function.

3. **Difference between Matplotlib and Seaborn.**
   *Answer*: 
   - **Matplotlib** is a foundational, low-level plotting library in Python that provides extensive flexibility and customization for creating static, animated, and interactive visualizations. However, it can be verbose.
   - **Seaborn** is a high-level interface built on top of Matplotlib. It provides a more aesthetically pleasing default style, is specifically tailored for statistical graphics (like violin plots or pair plots), and integrates closely with Pandas DataFrames, requiring significantly less code.
