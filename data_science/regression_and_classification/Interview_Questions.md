# Regression and Classification - Interview Questions

1. **What is Regression?**
   *Answer*: Regression is a supervised learning technique used to predict a continuous numerical output based on one or more input features (e.g., predicting house prices, temperature, or sales).

2. **What are the different types of Regression?**
   *Answer*: Common types include Simple Linear Regression, Multiple Linear Regression, Polynomial Regression, Ridge Regression, Lasso Regression, and Support Vector Regression.

3. **What is Correlation?**
   *Answer*: Correlation is a statistical measure that expresses the extent to which two variables are linearly related (meaning they change together at a constant rate). It ranges from -1 to +1.

4. **How the Gradient Descent Algorithm Works?**
   *Answer*: Gradient descent is an optimization algorithm used to minimize the cost (loss) function. It iteratively adjusts the model's parameters (weights and biases) by moving in the direction of the steepest descent, which is defined by the negative of the gradient of the cost function.

5. **What are the evaluation metrics used for Regression?**
   *Answer*: Key metrics include Mean Absolute Error (MAE), Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and R-squared ($R^2$).

6. **When to use Regression and Classification?**
   *Answer*: Use Regression when the target variable you want to predict is continuous (e.g., salary, price). Use Classification when the target variable is categorical or discrete (e.g., spam/not spam, disease/no disease).

7. **What are the evaluation metrics used for Classification?**
   *Answer*: Key metrics include Accuracy, Precision, Recall, F1-Score, Confusion Matrix, and ROC-AUC.

8. **When to use which metric?**
   *Answer*: 
   - **Accuracy**: When the dataset is balanced.
   - **Precision**: When false positives are highly costly (e.g., spam detection).
   - **Recall**: When false negatives are highly costly (e.g., cancer detection).
   - **F1-Score**: When you need a balance between Precision and Recall or when dealing with imbalanced datasets.

9. **What is balanced dataset?**
   *Answer*: A balanced dataset is one where the classes of the target variable are roughly equally distributed (e.g., 50% positive cases and 50% negative cases).

10. **What are the applications of Regression?**
    *Answer*: Stock price prediction, sales and revenue forecasting, real estate pricing, and risk assessment.

11. **What are the applications of Classification?**
    *Answer*: Email spam filtering, medical diagnosis, sentiment analysis, image classification, and fraud detection.
